# NutriSmart

**NutriSmart** is an open-source desktop application for intelligent dietary planning, calorie analysis, and wellness monitoring.  
It helps users create health profiles, set fitness goals, log meals, and receive automated calorie and nutrient analysis.  
The system generates personalized diet plans using BMI, activity level, and health parameters.

## Modules

- **UserAuth**: Registration, login, and user profile management  
- **DietPlan**: Diet planning and personalized recommendations  
- **FoodDB**: Food database management and nutritional data updates  
- **Visualization**: Dashboards and progress tracking

## Technologies

- Programming Language: Python  
- GUI: Tkinter / PyQt  
- Database: MySQL  
- Data Analysis: Pandas, Matplotlib  
- Tools: Git, PyCharm / VS Code
